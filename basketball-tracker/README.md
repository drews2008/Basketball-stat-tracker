# Basketball Stat Tracker

Tablet-friendly basketball stat tracker for high school games.

## Features

- Full-court display with 3PT line, free throw, keys, half-court
- Tap anywhere for 2pt/3pt shots, drag to adjust, confirm to save
- Input stats for all players
- View player & team stats in real time
- Export & import game data
